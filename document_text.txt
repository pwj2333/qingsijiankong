获取授权accessToken
接口功能：获取accessToken
接口说明：登录获取accessToken
接口请求地址：https://haiwang.kinthtech.com.cn/msaPlatform/platform/user/login
请求方式：POST
参数类型：x-www-from-urlencoded
请求参数说明：
字段名
字段说明
字段类型
是否必填
备注
account
账号
String
y
password
密码
String
y
调用的时候要做md5加密后给后端
响应字段说明：
字段名
类型
说明
accessToken
string
接口调用凭证（后续调用接口需要）
expiresIn
string
过期时间（默认2小时）
响应示例
成功响应编码：
{
"msg": "账号密码登录成功",
"item": {
"accessToken": "c0074a2d-8ae2-40c4-95ff-7c32a0d3b28f",
"expiresIn": 7200
},
"code": 100001,
"success": true,
"message": "账号密码登录成功"
}
失败响应编码：
{
"msg": "busy system",
"code": 100002,
"entity": "获取token失败！"
}
